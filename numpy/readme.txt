This excersise was done to learn the fundamentals of Numpy for data analysis. 
Key learnings include:
* **Array Creation:** Generation of arrays using `arange`, `zeros`, `ones`, and random number generators (`rand`, `randn`).
* **Matrix Operations:** Reshaping 1D arrays into 2D matrices and performing axis-based aggregations (sum, standard deviation, etc.).
* **Advanced Indexing:** Efficiently selecting, slicing, and extracting specific elements or sub-matrices from complex data structures.
* **Vectorized Math:** Performing element-wise arithmetic and utilizing universal functions (ufuncs) for optimized computation.
