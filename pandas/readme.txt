These projects focused on learning how to use Pandas library to perform Exploratory Data Analysis. 
Here are the key takeaways:

Data Wrangling: Expertise in cleaning and prepping large-scale CSV data for analysis.

Statistical Profiling: Calculating averages, medians, and outliers to define dataset boundaries.

Segment Analysis: Filtering data by multiple attributes (e.g., job title, spending habits, credit providers).

Dynamic Transformations: Creating custom data features using Python’s functional programming tools.

Trend Discovery: Aggregating data to find year-over-year changes and top-performing categories.
